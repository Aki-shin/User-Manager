Интеграция с GLPI

Приложение подключается к серверу GLPI по SSH и запускает встроенную CLI-команду glpi:ldap:synchronize_users. GLPI сам читает из FreeIPA и применяет изменения (создаёт новых, обновляет атрибуты, удаляет ушедших согласно настройке AuthLDAP «Поведение для удалённых в LDAP пользователей»).

Поле --ldap-server-id
Пароль SSH или passphrase для ключа
Если задан — используется ключ, не пароль
Плейсхолдер {auths_id} подставится автоматически. Флаг --no-interaction обязателен — иначе CLI зависнет на вопросе «Желаете продолжить?».
